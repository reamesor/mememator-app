# Mememator $MATE

**Solana meme-to-market engine.** Turn memes into tokens—easy launch to Pump.fun for launcher wannabes.

## What it does

- **Meme Generator** — Pick KOLs, themes, or upload your own image. Add text, pick a template. Download or send to Forge.
- **Token Lore & Meme** — Write the narrative, meme angle, and theme for your token. AI-improve with one click. Copy for pump.fun or send to Forge.
- **The Forge** — Combine meme images + lore. Fill token name, ticker, description. Burn $MATE and launch to Pump.fun.
- **Your folder** — Connect your wallet. Memes, lores, and launches are saved per wallet so you can reuse and pick from your history.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech

- Next.js 16, React 19
- Solana wallet adapter (Phantom, Solflare)
- Tailwind CSS
